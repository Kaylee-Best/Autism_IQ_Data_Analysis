# Autism_IQ_Data_Analysis
Student/Personal Project exploring the relationship between IQ and Autism diagnosis.

Question(s): How does an Autism Diagnosis relate to IQ? Are those on the Autism Spectrum more likely to have higher IQs or lower IQs when compared to a general population? Amongst Each other?

Note: The spreadsheet used for this project uses synthetic data and may not reflect actual trends in the population. This project is purely for exploratory and educational purposes.

Methodology: Spreadsheet downloaded from Kaggle, use of EDA via Python to explore columns, rows, categorical data, quantitative data, and data visualization.
